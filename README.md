# כלי מיון, קיבוץ ומתן-שמות למסמכים

כלי Streamlit עצמאי: מעלים קבצים (PDF/תמונות), הכלי קורא אותם, מקבץ מסמכים
לוגיים, נותן שם לפי כללים, ממזג כל קבוצה ל-PDF אחד, ומאפשר הורדה.
ההעלאה למונדיי נשארת ידנית.

## הרצה מקומית
```bash
pip install -r requirements.txt
streamlit run app.py
```
בסרגל הצד מזינים Anthropic API Key (או שומרים אותו ב-secrets).

## פרסום לאינטרנט (קבלת URL) – Streamlit Community Cloud
1. העלה את התיקייה (app.py, requirements.txt) ל-GitHub.
2. היכנס ל-https://share.streamlit.io , חבר את ה-repo ובחר את `app.py`.
3. תחת **Settings → Secrets** הוסף:
   ```
   ANTHROPIC_API_KEY = "sk-ant-..."
   ```
4. תקבל URL קבוע שאפשר לשלוח לעובדת.

## התאמות
- **כללי מתן-שמות**: עורכים ישירות בתיבה בסרגל הצד (או משנים את
  `DEFAULT_NAMING_RULES` בקוד לקביעת ברירת מחדל).
- **סף ביטחון**: מתחתיו מסמך מסומן "לבדיקה" במקום העלאה — נשלט בסליידר.
- **מודל**: `claude-sonnet-5` כברירת מחדל; אפשר `claude-opus-4-8` למסמכים קשים.

## מגבלות ידועות
- תומך ב-PDF, JPG, PNG, WEBP, GIF. קובצי HEIC (לפעמים מאייפון) דורשים המרה מראש.
- כל קובץ נשלח ל-API בנפרד, אז חבילה גדולה מאוד תיקח כמה עשרות שניות.
